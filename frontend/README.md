  <h2>Overview</h2>
  <p>A **multi-page Task Manager App** built to revise and strengthen React skills. Users can **add, edit, delete, and categorize tasks**, with data stored in **LocalStorage**. Backend integration is planned for MongoDB later.</p>

  <h2>Features</h2>
  <ul>
    <li>Add/Edit/Delete tasks</li>
    <li>Task categorization</li>
    <li>Set due dates & track status</li>
    <li>Multi-page navigation with React Router</li>
    <li>Responsive UI with CSS</li>
  </ul>

  <h2>Tech Stack</h2>
  <ul>
    <li>ReactJS</li>
    <li>React Router</li>
    <li>LocalStorage</li>
    <li>CSS</li>
    <li>Backend + MongoDB (coming soon)</li>
  </ul>

  <h2>Setup</h2>
  <pre><code>git clone https://github.com/Parthwebde12/taskmanager.git;
cd task-manager
npm install
npm start</code></pre>
